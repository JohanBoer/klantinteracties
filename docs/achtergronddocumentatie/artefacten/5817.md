# F5817 - Zoek partijidentificator

## Functionaliteit

Deze functie retourneert alle `partijidentificatoren` die aan de opgegeven zoekcriteria voldoen.

## API Calls

- get ../partijidentificatoren?objectId=999990019&soortObjectId=burgerservicenummer
