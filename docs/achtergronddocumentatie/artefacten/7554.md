# F7554 - Registreer partij

## Functionaliteit

De functie registreert een nieuwe `partij` en de daarbij behorende `Partijidentificator`. 

Vooralsnog kan de partijidentificator bestaan zonder dat er een gerelateerde partij is. Er kan dan echter geen verwijzing naar de partij (`identificeerdePartij`) opgenomen worden.


Een partij kan niet bestaan zonder dat er een partijidentificator bestaat. Aangezien er hier transactionele integriteit gewaarborgd moet worden 
is de volgorde van vastleggen van belang.

Eerst wordt de partijidentificator vastgelegd, zonder verwijzing naar een Partij.
